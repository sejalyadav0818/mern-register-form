# Registration  with React, Express, and MongoDB

This project provides a simple user registration form using React for the frontend, Express for the backend, and MongoDB as the database.

## Project Structure

- Frontend: React application
- Backend: Express application with MongoDB integration

## Prerequisites

- Node.js and npm installed
- MongoDB server running locally or a cloud instance

## Setup & Installation

### 1. Frontend Setup

Navigate to the frontend directory (assuming it's the root directory for now).

```bash
cd path/to/frontend
```

Install the necessary npm packages:

```bash
npm install
```

Run the React development server:

```bash
npm start
```

By default, it should run on `http://localhost:3000`.

### 2. Backend Setup

Navigate to the backend directory.

```bash
cd path/to/backend
```

Install the necessary npm packages:

```bash
npm install
```

Before running the backend server, ensure that MongoDB is running and accessible.

Run the Express server:

```bash
npm run start
```

By default, the server will run on `http://localhost:3001`.

## Features

- Registration form with fields: Username, Email, and Password.
- Data storage in MongoDB.
